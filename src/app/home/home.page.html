DEVALDI AKBAR SURYADI SICAL singkatan dari Simple Calculator, adalah aplikasi buatan penulis dengan tujuan membantu manusia dalam melakukan perhitungan standar yang saat ini hanya mampu seperti tambah, kurang, kali, bagi, persen, plus, minus serta menghapus nominal yang akan penulis kembangkan seiring berjalannya waktu. Aplikasi ini dibuat dengan bantuan framework Ionic versi 5 dengan bahasa Angular, serta penulis menggunakan code editor visual Studio Code.
